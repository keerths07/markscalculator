# AGENTS.md

## Project Information

Project Name: Marks Calculator

## Purpose

The Marks Calculator application allows users to calculate total marks, average marks, percentage, and grades based on marks entered for multiple subjects.

## Project Structure

* README.md - Project overview and setup instructions
* CONTRIBUTING.md - Contribution guidelines
* USER_MANUAL.md - User guide
* AGENTS.md - AI agent instructions

## Agent Instructions

When working on this project:

1. Maintain simple and readable code.
2. Preserve existing functionality.
3. Ensure calculations remain accurate.
4. Validate user inputs before processing.
5. Update documentation when adding features.
6. Follow the existing coding style.
7. Do not remove core calculation features without approval.

## Features

* Subject-wise marks entry
* Total marks calculation
* Average calculation
* Percentage calculation
* Grade generation
* Input validation

## Future Enhancements

* GPA calculation
* Export results to PDF
* Student performance analytics

Any modifications should prioritize accuracy, usability, and maintainability.
