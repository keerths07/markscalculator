# User Manual

## Overview

Marks Calculator is a simple application that helps students calculate:

* Total Marks
* Average Marks
* Percentage
* Grade

## How to Use

### Step 1: Enter Marks

Input marks obtained in each subject.

### Step 2: Calculate

Click the "Calculate" button.

### Step 3: View Results

The application will display:

* Total Marks
* Average Marks
* Percentage
* Grade

## Grade Criteria

| Percentage | Grade |
| ---------- | ----- |
| 90-100%    | A+    |
| 80-89%     | A     |
| 70-79%     | B     |
| 60-69%     | C     |
| 50-59%     | D     |
| Below 50%  | F     |

## Example

Subjects:

* Math: 95
* Science: 88
* English: 90

Result:

* Total: 273
* Average: 91
* Percentage: 91%
* Grade: A+

## Troubleshooting

### Invalid Input

Ensure marks are:

* Numeric values
* Within the allowed range (0-100)

### Calculation Errors

Refresh the application and re-enter the values.
